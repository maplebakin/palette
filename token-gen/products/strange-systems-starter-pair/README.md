# Strange Systems Starter Pair

Strange Systems Starter Pair is a two-pack bundle containing the Beef Ritual and Cobalt Chapel Apocapalette theme packs.

This bundle is intended as a starter set for designers, developers, and digital makers who want two complete, ready-to-use color systems with different moods:

- Beef Ritual: a dark burnt-red system with electric cyan accents and smoky surfaces.
- Cobalt Chapel: a saturated cobalt and chapel-dark navy system with teal patina, cool ivory, and candlelight gold.

## Included Files

- `Beef-Ritual-Theme-Pack.zip`
- `Cobalt-Chapel-Theme-Pack.zip`
- `README.md`
- `USAGE.txt`

Each theme pack ZIP contains its own customer README, CSS variables, canonical JSON tokens, Figma-friendly token JSON, Penpot-compatible token JSON, LibreOffice/OpenOffice palette, and SVG previews.
