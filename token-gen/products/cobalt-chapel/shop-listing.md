# Cobalt Chapel - Apocapalette Theme Pack

## Product Title

Cobalt Chapel - Apocapalette Theme Pack

## Short Description

Cobalt Chapel is a sacred-tech blue Apocapalette theme pack built for polished dashboards, dev tools, portfolio sites, gallery pages, music/nightlife brands, and bold digital products.

## Long Description

Cobalt Chapel is a complete Apocapalette theme pack built around saturated cobalt, chapel-dark navy surfaces, cool ivory text, teal patina accents, and restrained candlelight gold. It is designed for makers who want a blue system with atmosphere and authority instead of another flat corporate navy palette.

The pack includes CSS variables for web projects, structured JSON tokens, Figma-friendly token JSON, Penpot-compatible tokens, a LibreOffice/OpenOffice `.soc` palette, and preview SVGs for quick visual reference. Use it as a starting point for dashboards, developer tools, AI interfaces, finance concepts, portfolio sites, gallery pages, music/nightlife brands, or dramatic product prototypes.

It feels expensive, controlled, and a little strange: polished enough for production, distinctive enough to remember.
