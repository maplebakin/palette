# Beef Ritual - Apocapalette Theme Pack

Beef Ritual is a dark, strange, high-contrast Apocapalette theme pack for dramatic interfaces, game UI, occult dashboards, and product concepts that need a sharp visual identity.

## What You Get

- `Beef-Ritual-Theme-Pack.zip` - the complete theme pack
- `USAGE.txt` - plain-language usage and license notes
- `shop-listing.md` - draft listing copy for reference
- `tags.txt` - marketplace/search tags
- `preview/palette-card.svg` - product preview artwork
- `preview/swatch-strip.svg` - quick swatch reference

## Inside the Theme Pack ZIP

The ZIP includes CSS variables, JSON tokens, Figma token JSON, Penpot token JSON, a LibreOffice/OpenOffice `.soc` palette, README guidance, and SVG previews.

## Quick Start

1. Open `Beef-Ritual-Theme-Pack.zip`.
2. Use `css/variables.css` in web projects.
3. Use `tokens.json`, `figma/tokens.json`, or `penpot/tokens.json` in design-token workflows.
4. Import `libreoffice/beef-ritual.soc` into LibreOffice or OpenOffice when you need matching document colors.

Made with Apocapalette.
