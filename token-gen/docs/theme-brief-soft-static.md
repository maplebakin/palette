# Soft Static Theme Brief

## Theme Name

Soft Static

## One-Sentence Concept

A quiet digital haze theme built from pale neutrals, soft lavender-grey, muted blue, and gentle signal colors.

## Mood Keywords

- Soft noise
- Calm interface
- Cloud off-white
- Mist lavender
- Low-friction
- Editorial
- Desaturated
- Gentle system

## Ideal Use Cases

- Writing apps and knowledge bases
- Wellness and reflection tools
- Calm productivity dashboards
- Portfolio templates
- Documentation sites
- Editorial microsites
- Lightweight SaaS prototypes that need softness without beige sameness

## Palette Direction

Build a light-first system that still has enough structure for UI. Use cloud off-white, static grey, mist lavender, faded blue, muted sage, warm shell, and soft graphite.

Rough families:

- Cloud off-white and warm shell for main backgrounds
- Static grey and pale blue-grey for panels and borders
- Mist lavender for brand softness and quiet emphasis
- Faded blue for links and selected states
- Muted sage for success or calm confirmation
- Soft graphite for text and strong UI anchors
- Dusty coral or warm peach for rare alert accents

## Accessibility and Contrast Notes

- The main risk is washed-out contrast; keep text tokens assertive enough.
- Use soft color for surfaces, not for all text.
- Ensure focus, selected, and error states have clear value contrast.
- Include at least one darker surface or inverse token for previews and components that need weight.

## Token Personality Notes

- Brand: mist lavender or faded blue, soft but identifiable.
- Surface: cloud off-white, warm shell, and pale static grey.
- Accent: faded blue for links and interactive emphasis.
- Danger: dusty coral/red that stays legible without feeling harsh.
- Success: muted sage, calm and readable.
- Warning: soft amber or warm sand with enough contrast.
- Text: soft graphite for primary text, grey-lavender for muted text, off-white for inverse text.

## Suggested Preview and Listing Language

Soft Static is a calm Apocapalette theme pack for writing tools, knowledge bases, wellness products, and quiet dashboards. It uses cloud neutrals, mist lavender, faded blue, and soft graphite to create a tokenized palette that feels gentle without becoming vague.

## Avoid

- Do not let the palette become beige-only or grey-only.
- Do not use pale lavender for important text unless contrast is proven.
- Avoid spa-cliche mint and cream combinations.
- Avoid decorative gradients; this should feel quiet and system-ready.
- Avoid making all alerts too polite to notice.
